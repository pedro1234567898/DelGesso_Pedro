# TP1 — Productos Reales con Machine Learning

## 1. Netflix - Sistema de Recomendaciones
- **Nombre del producto/servicio**: Sistema de recomendaciones de Netflix.
- **Qué problema resuelve con ML**: Personaliza sugerencias de películas y series para mantener a los usuarios enganchados, aumentando la retención y el tiempo de visualización al predecir contenido que les gustará basado en patrones de consumo.
- **Qué tipo de ML usa y por qué**: Supervisado, porque se entrena con datos etiquetados como calificaciones y historiales de visualización de usuarios para predecir preferencias futuras. Utiliza modelos híbridos como filtrado colaborativo y redes neuronales profundas, que aprenden de ejemplos conocidos para hacer predicciones.
- **Qué datos probablemente necesita para funcionar**: Historial de visualizaciones de usuarios, calificaciones de contenido, metadatos de películas/series (género, actores, duración), y datos de interacción como pausas o rewinds.

## 2. Spotify - Recomendaciones de Música
- **Nombre del producto/servicio**: Sistema de recomendaciones de Spotify (ej. Discover Weekly).
- **Qué problema resuelve con ML**: Ayuda a los usuarios a descubrir nueva música y playlists personalizadas, resolviendo la sobrecarga de opciones en un catálogo masivo al analizar gustos y patrones de escucha.
- **Qué tipo de ML usa y por qué**: Supervisado, ya que emplea filtrado colaborativo y modelos de deep learning entrenados con datos etiquetados de escuchas pasadas para predecir canciones similares. Los modelos como redes neuronales convolucionales (CNN) clasifican y recomiendan basados en ejemplos de preferencias reales.
- **Qué datos probablemente necesita para funcionar**: Historial de reproducciones de usuarios, características de audio de canciones (tempo, género, energía), metadatos de tracks (artista, álbum), y datos de interacción como skips o adds to playlist.

## 3. Google Translate
- **Nombre del producto/servicio**: Google Translate.
- **Qué problema resuelve con ML**: Facilita la traducción precisa y natural entre idiomas, superando barreras lingüísticas en comunicaciones globales al procesar contextos completos en lugar de palabras aisladas.
- **Qué tipo de ML usa y por qué**: Supervisado, porque utiliza traducción neuronal (Neural Machine Translation) entrenada con pares de oraciones traducidas como datos etiquetados para aprender patrones y generar traducciones. Es supervisado ya que el modelo predice salidas basadas en entradas conocidas y corregidas.
- **Qué datos probablemente necesita para funcionar**: Corpora bilingües masivos (millones de oraciones traducidas por humanos), datos de contexto lingüístico, y retroalimentación de usuarios para refinar modelos.

## 4. Tesla Autopilot
- **Nombre del producto/servicio**: Tesla Autopilot (sistema de conducción asistida).
- **Qué problema resuelve con ML**: Permite una conducción semi-autónoma segura, reduciendo accidentes y fatiga del conductor al detectar objetos, planificar rutas y tomar decisiones en tiempo real.
- **Qué tipo de ML usa y por qué**: Por refuerzo, porque el sistema de planificación y decisión aprende de simulaciones y datos reales de conducción, maximizando recompensas como seguridad y eficiencia a través de pruebas y errores en entornos dinámicos. Combina con deep learning supervisado para percepción (ej. CNN para visión), pero el núcleo de autonomía es por refuerzo para adaptarse a escenarios impredecibles.
- **Qué datos probablemente necesita para funcionar**: Imágenes de cámaras, datos de sensores (radar, ultrasonido), trayectorias de conducción humana, datos de flota de vehículos Tesla, y simulaciones de escenarios de tráfico.

## 5. Mercado Libre - Sistema de Recomendaciones y Detección de Fraude
- **Nombre del producto/servicio**: Plataforma de e-commerce de Mercado Libre (con ML en recomendaciones y fraude).
- **Qué problema resuelve con ML**: Personaliza recomendaciones de productos para aumentar ventas y detecta fraudes en transacciones para proteger a usuarios y vendedores en un mercado digital masivo.
- **Qué tipo de ML usa y por qué**: Supervisado, ya que modelos como filtrado colaborativo y clasificación de productos se entrenan con datos etiquetados de compras pasadas y transacciones fraudulentas para predecir preferencias o riesgos. Es supervisado porque usa ejemplos conocidos (etiquetas de fraude o compras) para clasificar nuevos datos.
- **Qué datos probablemente necesita para funcionar**: Historial de navegación y compras de usuarios, metadatos de productos, datos de transacciones (dispositivo, patrones de comportamiento), y variables de riesgo como ubicaciones o montos inusuales.
